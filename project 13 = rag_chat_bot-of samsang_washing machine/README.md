# Project 13: RAG Chatbot for Samsung Washing Machine

## Overview

This project focuses on building an **AI-powered Retrieval-Augmented Generation (RAG) chatbot** for a **Samsung Washing Machine user support system**.

The chatbot helps users get accurate answers related to:
- Product specifications
- Installation instructions
- Washing modes and features
- Error codes
- Troubleshooting steps
- Maintenance guidelines
- User manual information


# Features

## Product Knowledge Assistant

The chatbot can answer questions about:

- Washing machine features
- Model specifications
- Capacity information
- Energy efficiency
- Smart features
- Warranty details
