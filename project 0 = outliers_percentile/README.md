# outliers_percentile

A simple utility for detecting outliers in a dataset using the percentile (IQR) method.

## Overview

`outliers_percentile` identifies values that fall outside a defined percentile range. It is useful for data cleaning, exploratory data analysis, and preprocessing before machine learning workflows.

The method uses percentile boundaries to detect unusually high or low values.

## Features

- Detect lower and upper percentile-based outliers
- Configurable percentile thresholds
- Returns outlier values and indexes
- Works with numerical datasets
- Lightweight and easy to integrate

Applications
Data preprocessing
Exploratory Data Analysis (EDA)
Removing noisy observations
Feature engineering
Statistical analysis
Requirements
Python >= 3.8
NumPy
